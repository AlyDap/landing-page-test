Menjadikan folder menjadi repo

git init

Melihat status repo

git status

Masuk ke File

cd nama file

Keluar dari File

cd ..

Melihat isi file

ls

Membersihkan tanpilan git bash

clear

untuk alias graph

alias graph="git log --all --decorate --oneline --graph"

untuk mentracked file baru ke repo

git add . => add semua file baru yang statusnya Untracked File

untuk membatalkan tracked file baru dari repo

git restore --staged . => pulihkan semua file ke status Untracked

untuk uplod file baru dari repo yang sudah di add

git commit -m "pesan commit" => untuk upload ke repo

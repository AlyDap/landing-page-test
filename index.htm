Menjadikan folder menjadi repo

git init

Melihat status repo

git status

Masuk ke File

cd nama file

Keluar dari File

cd ..

Melihat isi file

ls

Membersihkan tanpilan git bash

clear

untuk alias graph

alias graph="git log --all --decorate --oneline --graph"

untuk mentracked file baru ke repo

git add . => add semua file baru yang statusnya Untracked File

untuk membatalkan tracked file baru dari repo

git restore --staged . => pulihkan semua file ke status Untracked

untuk uplod file baru dari repo yang sudah di add

git commit -m "pesan commit" => untuk upload ke repo

untuk lihat username, email, dll.

git config --list => bisa lihat detail list

Ganti usernam dan email GIT

git config --global user.name "nama username"

git config --global user.email "email@gmail.com"

=> Ganti user dan password secara global di seluruh folder PC

git config --local user.name "nama username"

git config --local user.email "email@gmail.com"

=> Ganti user dan password secara lokal / hanya di folder yang dipilih

git help
git help -a
git help -g
press q to exit / return

Menjadikan folder menjadi repo

git init

Melihat status repo

git status

Masuk ke File

cd nama file

Keluar dari File

cd ..

Melihat isi file

ls

Membersihkan tampilan git bash

clear

untuk alias graph

alias graph="git log --all --decorate --oneline --graph"

untuk mentracked file baru ke repo

git add . => add semua file baru yang statusnya Untracked File

untuk membatalkan tracked file baru dari repo

git restore --staged . => pulihkan semua file ke status Untracked

untuk uplod file baru dari repo yang sudah di add

git commit -m "pesan commit" => untuk upload ke repo

Modifeid file yang sudah ada

git commit -am "pesan commit" => untuk updated file di repo

Membuat folder dari git bash

mkdir nama_file

Untuk remote file yang sudah ada di git-hub tapi belum ada di penyimpanan local

git clone https://github.com/AlyDap/landing-page-test.git

Bisa juga langsung di VSCode dengan klik Ctrl+Shift+P

Untuk update file di gitbash yang suda di edit di penyimpanan local

git push

Untuk remote file yang sudah ada di local penyimpanan tapi belum ada di git-hub

Harus buat folder / file kosong di penyimpanan local

Lalu buat repo kosong di git-hub

Kemudian ketik code di bawah untuk meremotenya

git remote add origin https://github.com/AlyDap/landing-page-test.git

Untuk push ke gitbash dari penyimpanan local khusus repo kosong

git branch -M main

git push -u origin main

Untuk Cek sudah sampai mana repo di git-hub

git fetch

Untuk Merge file yang tidak bisa dipush

git pull

Lalu di add & commit seperti biasa

Kemudian di push lagi

untuk lihat username, email, dll.

git config --list => bisa lihat detail list

Ganti username dan email GIT

git config --global user.name "nama username"

git config --global user.email "email@gmail.com"

=> Ganti user dan password secara global di seluruh folder PC

git config --local user.name "nama username"

git config --local user.email "email@gmail.com"

=> Ganti user dan password secara lokal / hanya di folder yang dipilih

git help
git help -a
git help -g
press q to exit / return
